# n8n-nodes-syphoon

Generic web scraping node for n8n similar to ScrapingBee.

## Features
- Scrape any URL
- Auto detect product/listing/article
- Sitemap scraping
- SERP scraping
- Structured output

## Credentials
Create API key at:
https://app.syphoon.com

## Example
Add Syphoon node → Enter URL → Run
